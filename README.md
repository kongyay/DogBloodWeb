# DogBloodWeb

## Configuration

CLIENT

```
- Edit 'client_dog/.env'
```

SERVER

```
- Rename 'server_dog/prod_example.cfg' to 'server_dog/prod.cfg'
- Edit 'server_dog/prod.cfg'
```

## Project setup

CLIENT

```
cd client_dog
yarn install
```

SERVER

```
cd server_dog
virtual\Scripts\activate
pip install -r virtual/requirements.txt
python app/main.py
```

### Compiles and hot-reloads for development

CLIENT

```
yarn run serve
```
